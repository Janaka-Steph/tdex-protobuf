Protocol Buffers spec defintions for TDEX

## Requirements

* protoc
* Node/npm (for typescript plugin)
* Go

## Compile

Build stubs for JavaScript & TypeScript

```sh
$ make js
```

Build for Go

```sh
$ make go
```

## Usage

* JavaScript

```sh
$ npm install tdex-network/tdex-protobuf
```

* Go 

```sh
$ go get github.com/tdex-network/tdex-protobuf
```
